## Lineage Findings

- Frozen contracts: `research/research_latest.json`, `research/strategy_matrix.csv`, and public schemas in `packages/qre_artifacts/public_outputs.py`.
- Sidecar artifacts: source resolution, source qualifications, Tiingo E2E logs, campaign evidence ledger, research memory, routing/sampling/preflight reports.
- Orphan producers: `logs/qre_tiingo_hypothesis_generator_e2e/latest.json` is a working producer artifact with tests and docs but no canonical candidate/screening/memory consumer found.
- Orphan artifacts: many static `logs/qre_*` report paths are producer/report surfaces with no proven later-run consumer from static inspection.
- Consumers assuming missing artifacts: `packages.qre_research.research_memory` has a broad default artifact list and fails closed on missing entries; some expected logs may not exist locally.
- Context-only authoritative-looking artifacts: paper/shadow/live README scaffolds and many readiness reports may look like gate authority, but their current authority is documentation or read-only reporting only.
