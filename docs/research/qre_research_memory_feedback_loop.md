# QRE Research Memory Feedback Loop

The governed research memory feedback loop consumes offline artifact summaries,
multi-window evidence closure outcomes, and canonical rejection reasons to
explain future hypothesis prioritization.

Negative evidence suppresses retesting unless conditions change. Missing
evidence creates a next action rather than a permanent rejection. Source/data
quality, architecture, maturity, policy, and operator-decision blockers remain
blockers.

The loop emits auditable prioritization records and do-not-retest lists. It does
not create production strategies or campaigns, bypass budgets or governance
gates, or grant synthesis, shadow, paper, live, broker, risk, order, or capital
authority.
