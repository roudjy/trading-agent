# QRE Evidence Memory Accumulation

This surface accumulates evidence and memory feedback across governed offline
dry-run and candidate-batch results.

It preserves:

- rejection reasons
- missing-vs-negative evidence distinction
- repeated failure modes
- hypothesis lineage
- disposition trends
- memory feedback records
- next-action queues
- retest suppression guidance

It does not create production artifacts or grant synthesis, shadow, paper,
live, broker, risk, order, or capital authority.
